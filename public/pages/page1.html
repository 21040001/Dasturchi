Java Tarixi

Java dasturlash tili tarixi juda qiziqarli. Java dastlab interaktiv televizorlar uchun mo'ljallangan edi, ammo o'sha paytdagi raqamli kabel televideniyesi uchun juda ilg'or texnologiya edi. Java dasturi 1991-yilda James Gosling, Mike Sheridan va Patrick Naughton tomonidan boshlangan va "Green Team" deb nomlangan kichik jamoa tomonidan ishlab chiqilgan. Dastlab, u kichik elektron qurilmalar, masalan, set-top boxlar uchun mo'ljallangan edi. Dastlab, u "Greentalk" deb nomlangan va fayl kengaytmasi edi. Keyinchalik, u "Oak" deb nomlandi va Green loyihasining bir qismi sifatida ishlab chiqilgan.

1995-yilda Oak "Java" deb qayta nomlandi, chunki Oak Technologies tomonidan savdo belgisi sifatida ro'yxatdan o'tgan edi. Java nomi Indoneziyaning Java orolidan olingan bo'lib, bu yerda birinchi marta qahva ishlab chiqarilgan (Java qahvasi deb ataladi). Java dasturi 1995-yilda Sun Microsystems tomonidan ommaga taqdim etilgan va "Bir marta yozing, hamma joyda ishlating" shiori bilan tanilgan . Java dasturi 1996-yilda birinchi rasmiy versiyasi JDK 1.0 chiqarildi.

Java dasturi hozirda internet dasturlash, mobil qurilmalar, o'yinlar, e-biznes yechimlari va boshqa ko'plab sohalarda keng qo'llaniladi.
