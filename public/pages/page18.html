Interfeyslar

Interfeyslar, obyektga yo'naltirilgan dasturlashda, sinflar tomonidan amalga oshirilishi kerak bo'lgan metodlar to'plamini belgilash uchun foydalaniladi. Interfeyslar yordamida sinflar o'rtasida mustahkam bog'lanishlar yaratmasdan turib, bir xil metodlarni turli xil sinflarda amalga oshirish mumkin.

Java'da interfeyslarni qanday amalga oshirish haqida oddiy bir misol keltiraman: 


        public interface Hayvon {
            void ovozBer();
        }

        public class It implements Hayvon {
            @Override
            public void ovozBer() {
                System.out.println("It vov-vov qiladi.");
            }
        }
        
        public class Mushuk implements Hayvon {
            @Override
            public void ovozBer() {
                System.out.println("Mushuk miyovlaydi.");
            }
        }

        public class Main {
            public static void main(String[] args) {
                Hayvon it = new It();
                Hayvon mushuk = new Mushuk();
                
                it.ovozBer();   // It vov-vov qiladi
                mushuk.ovozBer(); // Mushuk miyovlaydi
            }
        }

Ushbu misolda, Hayvon interfeysi umumiy metodni o'z ichiga oladi. It va Mushuk sinflari esa, Hayvon interfeysini amalga oshirib, ovozBer metodini o'ziga xos xatti-harakat bilan ta'minlaydi. Interfeyslar yordamida, bir xil metodlarni turli xil sinflarda amalga oshirish va mustahkam bog'lanishlardan qochish mumkin.
