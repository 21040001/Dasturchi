Java Sikllar

Sikl (Loop) nima?

Sikl yoki loop - bu dasturlashda takroriy amallarni bajarish uchun ishlatiladigan dasturiy konstruktsiya. Sikllar dasturdagi kod bloklarini bir necha marta qayta ishlatishga imkon beradi.

1. for Sikli

for sikli takroriy amallarni bajarish uchun ishlatiladi va oldindan belgilangan mantiqiy shart asosida ishlaydi.


        for (int i = 0; i < 5; i++) {
            System.out.println("i: " + i);
        }
2. while Sikli

while sikli shart true bo'lganicha takroriy amallarni bajaradi.


        int i = 0;
        while (i < 5) {
            System.out.println("i: " + i);
            i++;
        }
3. do-while Sikli

do-while sikli kamida bir marta bajariladi va shart keyin tekshiriladi.


        int i = 0;
        do {
            System.out.println("i: " + i);
            i++;
        } while (i < 5);
4. Misol: 1 dan 5 gacha sonlarni chiqarish

Quyidagi misolda 1 dan 5 gacha sonlarni chiqarish uchun turli sikl operatorlari ishlatilgan.


        // for sikli bilan
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }

        // while sikli bilan
        int j = 1;
        while (j <= 5) {
            System.out.println(j);
            j++;
        }

        // do-while sikli bilan
        int k = 1;
        do {
            System.out.println(k);
            k++;
        } while (k <= 5);
