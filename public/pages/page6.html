Java Operatorlari

Java dasturlash tilida operatorlar dasturda turli xil amallarni bajarish uchun ishlatiladi. Ular arifmetik, taqqoslash, mantiqiy, bitli va boshqa turdagi amallarni bajarishga yordam beradi.

1. Arifmetik Operatorlar

Arifmetik operatorlar sonlar ustida matematik amallarni bajarish uchun ishlatiladi.


        int a = 10;
        int b = 3;
        System.out.println(a + b); // 13
        System.out.println(a - b); // 7
        System.out.println(a * b); // 30
        System.out.println(a / b); // 3
        System.out.println(a % b); // 1
2. Taqqoslash Operatorlari

Taqqoslash operatorlari ikkita qiymatni solishtirish uchun ishlatiladi. Ular boolean (true yoki false) qiymat qaytaradi.


        int x = 10;
        int y = 20;
        System.out.println(x > y); // false
        System.out.println(x == y); // false
        System.out.println(x != y); // true
3. Mantiqiy Operatorlar

Mantiqiy operatorlar boolean qiymatlar ustida amallar bajaradi. Ular shartlarni birlashtirish uchun ishlatiladi.


        boolean a = true;
        boolean b = false;
        System.out.println(a && b); // false
        System.out.println(a || b); // true
        System.out.println(!a); // false
4. Bitli Operatorlar

Bitli operatorlar sonlarni ikkilik (binary) ko'rinishda ishlaydi. Ular bit darajasida amallar bajaradi.


        int a = 5; // 0101
        int b = 3; // 0011
        System.out.println(a & b); // 0001 -> 1
        System.out.println(a | b); // 0111 -> 7
        System.out.println(a ^ b); // 0110 -> 6
5. Tayinlash Operatorlari

Tayinlash operatorlari o'zgaruvchilarga qiymat berish uchun ishlatiladi.


        int a = 10;
        a += 5; // a = 15
        a *= 2; // a = 30
        System.out.println(a); // 30
6. Shartli (Ternary) Operator

Shartli operator (? :) shartga qarab ikki qiymatdan birini tanlash uchun ishlatiladi.


        int a = 10;
        int b = 20;
        int max = (a > b) ? a : b; // max = 20
        System.out.println(max);
7. Instanceof Operatori

instanceof operatori ob'ektning ma'lum bir sinfga tegishli ekanligini tekshiradi.


        String text = "Hello";
        System.out.println(text instanceof String); // true
8. Operatorlar Ustuvorligi

Operatorlar ustuvorligi (precedence) amallarning qaysi tartibda bajarilishini belgilaydi.


        int result = 10 + 5 * 2; // 5 * 2 = 10, keyin 10 + 10 = 20
        System.out.println(result); // 20
9. Qo'shimcha Operatorlar

Matnlarni birlashtirish va tur o'zgartirish (type casting) kabi amallar.


        String name = "Ali";
        String greeting = "Hello, " + name; // "Hello, Ali"

        double a = 10.5;
        int b = (int) a; // b = 10
