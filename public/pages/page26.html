Java Regex (Regular Expression) Tushuntirish

1. Regex nima?

Regex — bu matndan ma’lum bir strukturani aniqlash yoki tekshirish uchun ishlatiladigan ifoda.

Misollar:

  • Email: ^[\\w.-]+@[\\w.-]+\\.[a-zA-Z]{2,}$
  • Faqat raqam: ^\\d+$
  • Telefon raqami: ^\\+998\\d{9}$

2. Java’da Regex ishlatish

String.matches()

String s = "abc123";
boolean mos = s.matches("[a-z]+\\d+"); // true

Pattern va Matcher

import java.util.regex.*;

public class TestRegex {
    public static void main(String[] args) {
        String text = "Kod: AB1234";
        Pattern pattern = Pattern.compile("[A-Z]{2}\\d{4}");
        Matcher matcher = pattern.matcher(text);

        if (matcher.find()) {
            System.out.println("Topildi: " + matcher.group()); // AB1234
        }
    }
}

4. Amaliy misollar

Faqat raqamlar:

String s = "123456";
System.out.println(s.matches("\\d+")); // true

Email tekshirish:

String email = "user123@mail.com";
System.out.println(email.matches("^[\\w.-]+@[\\w.-]+\\.[a-zA-Z]{2,}$")); // true

Telefon raqami:

String tel = "+998901234567";
System.out.println(tel.matches("^\\+998\\d{9}$")); // true

Kuchli parol (katta harf va raqam):

String parol = "Java2023";
System.out.println(parol.matches("^(?=.*[A-Z])(?=.*\\d).{8,}$")); // true

5. Almashtirish va ajratish

replaceAll:

String s = "Salom123Dunyo";
String result = s.replaceAll("\\d", "*"); // Salom***Dunyo

split:

String s = "java,python,c++";
String[] t = s.split(","); // [java, python, c++]
