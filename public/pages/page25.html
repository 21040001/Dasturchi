Java Kalit So'zlari

public kalit so'zi:

Bu kalit so'zi sinfdagi o'zgaruvchilarga va metodlarga tashqi sinflardan, hatto boshqa paketlardan ham erishish imkonini beradi. Har bir kishi bu resurslarga to'liq kirish huquqiga ega bo'ladi. Masalan: public int age;

private kalit so'zi:

Bu kalit so'zi sinfdagi o'zgaruvchilar va metodlarga faqat shu sinf ichidan murojaat qilish imkonini beradi. Boshqa sinflardan yoki paketlardan bu resurslarga kirish mumkin emas. Masalan: private String name;

protected kalit so'zi:

Bu kalit so'zi faqat shu sinf va uning vorislaridan (subclass) foydalanishga imkon beradi. Bundan tashqari, bir xil paketdagi sinflar ham foydalanishi mumkin. Masalan: protected int salary;

Kalit So'zlarning Taqqoslanishi

Kalit So'z Xuddi Shu Sinf Xuddi Shu Paket Voris Sinf (Subclass) Tashqi Sinf (Boshqa Paket)
public ✅ ✅ ✅ ✅
protected ✅ ✅ ✅ ❌ (Agar voris bo‘lmasa)
no modifier (default) ✅ ✅ ❌ ❌

Misollar

public kalit so'zi:

        
        public class Person {
            public String name; // public o'zgaruvchi
            
            public void setName(String name) { // public metod
                this.name = name;
            }
        }

        public class Main {
            public static void main(String[] args) {
                Person p = new Person();
                p.setName("Ali");
                System.out.println(p.name);
            }
        }

private kalit so'zi:

        
        public class Person {
            private String name; // private o'zgaruvchi

            public void setName(String name) { // public metod
                this.name = name;
            }

            public String getName() { // public metod
                return this.name;
            }
        }

        public class Main {
            public static void main(String[] args) {
                Person p = new Person();
                p.setName("Ali");
                System.out.println(p.getName()); // private o'zgaruvchiga public metod orqali murojaat
            }
        }

protected kalit so'zi:

        
        public class Employee {
            protected String position; // protected o'zgaruvchi
        }

        public class Manager extends Employee {
            public void setPosition(String position) {
                this.position = position;
            }
        }

        public class Main {
            public static void main(String[] args) {
                Manager m = new Manager();
                m.setPosition("Manager");
                System.out.println(m.position); // protected o'zgaruvchiga voris sinfdan murojaat
            }
        }
