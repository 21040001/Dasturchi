Meros Olish

Java'da meros olish (inheritance), bir sinifin (class) başqa bir siniften özellik ve davranışları devralmasına olanak tanıyan bir nesne yönelimli programlama konseptidir. Meros olish orqali, kodni qayta ishlatish imkoniyati ortadi va sinflar o'rtasida o'xshash xatti-harakatlar bilan bog'liq bo'lgan mantiqiy bog'liqliklarni ifodalash mumkin bo'ladi.

Java'da meros olishni qanday amalga oshirish haqida oddiy bir misol keltiraman: 


        public class Hayvon {
            private String ism;
            
            public void setIsm(String ism) {
                this.ism = ism;
            }
            
            public String getIsm() {
                return ism;
            }
            
            public void ovozBer() {
                System.out.println("Hayvon tovush chiqaradi.");
            }
        }
        
        public class It extends Hayvon {
            @Override
            public void ovozBer() {
                System.out.println("It vov-vov qiladi.");
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                It it = new It();
                it.setIsm("Rex");
                System.out.println("Itning ismi: " + it.getIsm());
                it.ovozBer();
            }
        }

Ushbu misolda, Hayvon sinfi umumiy xususiyat va metodlarni o'z ichiga olgan super sinfdir. It sinfi esa, Hayvon sinfidan meros olib, ovozBer metodini qayta belgilaydi (override qiladi) va o'ziga xos xatti-harakatni ifodalaydi.
