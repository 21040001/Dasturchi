Metod Overloading

Metod overloading, obyektga yo'naltirilgan dasturlashda, bir sinfda bir xil nomga ega bo'lgan, ammo turli xil parametrlarga ega bo'lgan bir nechta usullarni belgilashni anglatadi. Bu dasturchilarga bir xil nomni turli xil vazifalar uchun ishlatish imkonini beradi.

Java'da metod overloadingni qanday amalga oshirish haqida oddiy bir misol keltiraman: 


        public class MathUtils {
            public int add(int a, int b) {
                return a + b;
            }
            
            public double add(double a, double b) {
                return a + b;
            }
            
            public int add(int a, int b, int c) {
                return a + b + c;
            }
        }

        public class Main {
            public static void main(String[] args) {
                MathUtils math = new MathUtils();
                System.out.println("2 + 3 = " + math.add(2, 3));
                System.out.println("2.5 + 3.5 = " + math.add(2.5, 3.5));
                System.out.println("1 + 2 + 3 = " + math.add(1, 2, 3));
            }
        }

Ushbu misolda, MathUtils sinfi bir nechta add usulini o'z ichiga oladi. Har bir add usuli turli xil parametrlarga ega va shunga mos tarzda turli xil arifmetik amallarni bajaradi. Metod overloading yordamida, bir xil nomdagi usullardan turli xil parametrlar bilan foydalanish va kodni sodda va tushunarli qilish mumkin.
