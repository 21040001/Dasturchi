Java Shart Operatorlari

Java dasturlash tilida if, else-if, else va switch-case shart operatorlari dasturda shartli amallarni bajarish uchun ishlatiladi.

1. if Operatori

if operatori shartni tekshiradi va shart true bo'lsa, unga tegishli kod blokini bajaradi.


        int a = 10;
        if (a > 5) {
            System.out.println("a 5 dan katta");
        }
2. if-else Operatori

if-else operatori shartni tekshiradi. Agar shart true bo'lsa, if bloki bajariladi, aks holda else bloki bajariladi.


        int a = 3;
        if (a > 5) {
            System.out.println("a 5 dan katta");
        } else {
            System.out.println("a 5 dan kichik yoki teng");
        }
3. if-else if-else Operatori

if-else if-else operatori bir nechta shartlarni ketma-ket tekshiradi. Birinchi true bo'lgan shartga tegishli blok bajariladi.


        int a = 7;
        if (a > 10) {
            System.out.println("a 10 dan katta");
        } else if (a > 5) {
            System.out.println("a 5 dan katta, lekin 10 dan kichik yoki teng");
        } else {
            System.out.println("a 5 dan kichik yoki teng");
        }
4. Ichma-ich if Operatori

if operatori ichida yana bir if operatori ishlatish mumkin. Bu ichma-ich shartlarni tekshirish uchun ishlatiladi.


        int a = 15;
        if (a > 10) {
            if (a < 20) {
                System.out.println("a 10 dan katta va 20 dan kichik");
            }
        }
5. Qisqa if-else (Ternary Operator)

Qisqa shartli operator (? :) shartga qarab ikki qiymatdan birini tanlash uchun ishlatiladi.


        int a = 10;
        int b = 20;
        int max = (a > b) ? a : b; // max = 20
        System.out.println(max);
6. switch-case Operatori

switch operatori bir nechta shartlarni tekshirish va bajarish uchun ishlatiladi. Har bir shart case bilan belgilanadi va kerakli kod bloklari amalga oshiriladi.


        int haftaKuni = 3;
        switch (haftaKuni) {
            case 1:
                System.out.println("Dushanba");
                break;
            case 2:
                System.out.println("Seshanba");
                break;
            case 3:
                System.out.println("Chorshanba");
                break;
            case 4:
                System.out.println("Payshanba");
                break;
            case 5:
                System.out.println("Juma");
                break;
            case 6:
                System.out.println("Shanba");
                break;
            case 7:
                System.out.println("Yakshanba");
                break;
            default:
                System.out.println("Noma'lum kun");
                break;
        }
7. Misol: Baholarni Tekshirish

Quyida talabaning bahosiga qarab uning natijasini chiqaruvchi misol keltirilgan.


        int baho = 85;
        if (baho >= 90) {
            System.out.println("A'lo");
        } else if (baho >= 80) {
            System.out.println("Yaxshi");
        } else if (baho >= 70) {
            System.out.println("Qoniqarli");
        } else {
            System.out.println("Qoniqarsiz");
        }
