Java'da Massivlar (Arrays) - To‘liq O‘zbekcha Darslik

1. Massiv nima?

Java dasturlash tilida massiv bu bir xil turdagi (masalan, faqat int yoki faqat String) bir nechta qiymatlarni ketma-ket tartibda saqlash uchun ishlatiladigan konteynerdir. Har bir qiymatga indeks yordamida murojaat qilinadi. Indeks har doim 0 dan boshlanadi.

Massivlar asosan quyidagi holatlar uchun juda foydali:

  • Ko‘p sonli o‘xshash qiymatlarni saqlash kerak bo‘lsa
  • Elementlarni tartibda saqlash va ularga tezda murojaat qilish zarur bo‘lsa

2. Java'dagi Massiv Turlari

Quyidagi jadvalda Java'da mavjud bo‘lgan asosiy massiv turlari va ularning misollari berilgan:

Massiv turi Izoh Misol
Bir o‘lchamli massiv Oddiy, tekis massiv. Har bir element indeks orqali chaqiriladi. int[] sonlar = {1, 2, 3};
Ikki o‘lchamli massiv Jadval yoki matritsa ko‘rinishida. Satr va ustunlar mavjud. int[][] matritsa = {{1,2},{3,4}};
Ko‘p o‘lchamli massiv 3 yoki undan ortiq o‘lchovli massivlar. Har bir o‘lcham bir ichki qatlamni bildiradi. int[][][] uchOlam = new int[2][3][4];
Jagged massiv Har bir satrda turli uzunlikdagi massivlar bo‘lishi mumkin. int[][] jagged = new int[2][];
Obyekt massivlari Elementlar oddiy son emas, balki obyektlardan iborat. String[] ismlar = {"Ali", "Vali"};

3. Bir o‘lchamli massiv

Massiv e'lon qilinadi, unga qiymatlar beriladi va elementlar konsolga chiqariladi:

int[] sonlar = new int[3]; // 3 ta bo‘sh elementli massiv yaratildi
sonlar[0] = 10;
sonlar[1] = 20;
sonlar[2] = 30;

// For sikli orqali massiv elementlari ekranga chiqariladi
for (int i = 0; i < sonlar.length; i++) {
    System.out.println("Element " + i + ": " + sonlar[i]);
}

4. Ikki o‘lchamli massiv (matritsa)

Ikki o‘lchamli massiv, ya'ni jadval yoki matritsa tuziladi va elementlar o‘qiladi:

int[][] matritsa = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println(matritsa[1][2]); // 2-satr, 3-ustundagi element (6)

5. Uch o‘lchamli massiv

Bu massiv "katta quti ichida qutilar" degan manoda, 3D tarzda tuziladi.

int[][][] uchOlam = new int[2][3][4];

// 1-blok, 3-qatordagi 4-elementga 99 ni yozamiz
uchOlam[0][2][3] = 99;
System.out.println(uchOlam[0][2][3]); // Natija: 99

6. Jagged massiv (notekis)

Har bir satrda har xil uzunlikdagi elementlar bo‘lgan massivlar quyidagicha:

int[][] jagged = new int[2][]; // 2 satr, lekin ustunlar aniqlanmagan
jagged[0] = new int[2]; // 1-satrga 2 ustun
jagged[1] = new int[4]; // 2-satrga 4 ustun

jagged[1][3] = 50;
System.out.println(jagged[1][3]); // Natija: 50

7. Obyekt massivlari

String massiv:

String[] ismlar = {"Ali", "Vali", "Gulnoza"};
System.out.println(ismlar[1]); // Natija: Vali

Custom sinf (class) massiv:

class Student {
    String name;

    Student(String name) {
        this.name = name;
    }
}

public class Main {
    public static void main(String[] args) {
        Student[] talabalar = new Student[2];
        talabalar[0] = new Student("Ali");
        talabalar[1] = new Student("Dilnoza");

        System.out.println(talabalar[1].name); // Dilnoza
    }
}

8. Foydali metodlar (Arrays klassi)

  • massiv.length – uzunlikni beradi
  • Arrays.sort(massiv) – elementlarni tartiblaydi
  • Arrays.copyOf(massiv, n) – nusxa oladi
  • Arrays.equals(m1, m2) – ikki massiv tengligini tekshiradi

Xulosa

Java dasturlash tilida massivlar yordamida ko‘plab ma’lumotlarni tartiblangan holatda saqlash, ularga tezda kirish va ular ustida turli amallar bajarish mumkin. Har xil turdagi massivlar – oddiydan to ko‘p o‘lchamligacha – turli ehtiyojlar uchun moslashgan.
