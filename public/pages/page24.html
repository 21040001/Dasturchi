Java'da ArrayList - Nazariy va Amaliy To‘liq Tushuntirish

1. ArrayList nima?

ArrayList — Java dasturlash tilida Collection Framework ning bir qismi bo‘lib, o‘lchami avtomatik kengayadigan yoki qisqaradigan dasturiy massiv turidir.

Oddiy massiv (Array) Java'da belgilangan o‘lchamga ega. U yaratib bo‘lingandan keyin hajmini o‘zgartirib bo‘lmaydi.
ArrayList esa, kerak bo‘lsa o‘zi avtomatik kengayadi, kerak bo‘lsa qisqaradi.

2. Array va ArrayList farqlari

Array ArrayList
O'lchami qat'iy, o'zgarmaydi O'lchami o'zgaruvchi (dynamic)
Oddiy malumot turlari (int, double) uchun ishlaydi Faqat obyektlar bilan ishlaydi (Integer, String, va h.k.)
Elementlarga indeks bilan murojaat Elementlarga indeks bilan ham, metodlar bilan ham ishlov beriladi
Tizim resursini kam ishlatadi Qulayliklar uchun ko'proq resurs talab qiladi

3. ArrayList yaratish

ArrayList ishlatishdan oldin uni import java.util.ArrayList; orqali chaqirish kerak. 

import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> mevalar = new ArrayList<>();
    }
}

Yuqoridagi kodda ArrayList<String> tipi ko'rsatilgan. Bu mevalar ro'yxati faqat String turlarini saqlashini bildiradi.

4. ArrayList metodlari

ArrayListda ishlatiladigan asosiy metodlar:

  • add(element) – yangi element qo‘shadi
  • get(index) – indeksdagi elementni qaytaradi
  • set(index, element) – indeksdagi elementni almashtiradi
  • remove(index yoki element) – elementni olib tashlaydi
  • clear() – barcha elementlarni o‘chiradi
  • size() – elementlar sonini qaytaradi
  • isEmpty() – bo‘sh yoki bo‘lmasligini tekshiradi
  • contains(element) – element bor-yo‘qligini tekshiradi

5. ArrayList bilan amaliy ishlash

import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> mevalar = new ArrayList<>();

        // Elementlar qo'shish
        mevalar.add(\"Olma\");
        mevalar.add(\"Banan\");
        mevalar.add(\"Anor\");

        // Elementlarni chiqarish
        for (String meva : mevalar) {
            System.out.println(meva);
        }

        // Elementni olib tashlash
        mevalar.remove(\"Banan\");

        // Yangilangan ro'yxat
        System.out.println(\"Yangi ro'yxat:\");
        for (String meva : mevalar) {
            System.out.println(meva);
        }
    }
}

6. Nima uchun ArrayList kerak?

Quyidagi holatlarda ArrayList foydalidir:

  • Ro‘yxatdagi elementlar soni oldindan noma'lum bo‘lsa
  • Tez-tez element qo‘shish va o‘chirish zarurati bo‘lsa
  • Elementlar ustida ko‘p amallar (qidirish, sortlash) bajarish kerak bo‘lsa
  • Kengaytiriladigan dinamik tuzilmalar kerak bo‘lsa

7. E'tibor bering

  • ArrayList faqat obyektlar saqlaydi. Agar siz son saqlamoqchi bo'lsangiz, Integer, Double kabi classlardan foydalanasiz (primitive tiplar emas).
  • ArrayList ichidagi elementlar tartibni saqlaydi (elementlar ketma-ket joylashadi).
  • Indeks orqali istalgan elementga tez kirish mumkin.

Xulosa

Java'da ArrayList — bu oson, qulay va kuchli dinamik massiv alternativasidir. Uning yordamida dasturda ro'yxat bilan ishlash osonlashadi, ma'lumot qo‘shish, o‘chirish va yangilash tez amalga oshiriladi.

O'rta va yirik loyihalarda massiv o‘rniga ko‘pincha aynan ArrayList ishlatiladi.
