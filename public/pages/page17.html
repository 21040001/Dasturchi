Abstraktsiya

Abstraktsiya, obyektga yo'naltirilgan dasturlashda, muhim bo'lgan xususiyatlarni ajratib olib, ortiqcha tafsilotlarni yashirishni anglatadi. Bu dasturlash konsepti yordamida murakkab tizimlar sodda va tushunarli shaklda taqdim etiladi.

Java'da abstraktsiyani amalga oshirish uchun abstract sinflar va interfeyslardan foydalanish mumkin. Oddiy bir misol keltiraman: 


        public abstract class Transport {
            private String model;
            
            public String getModel() {
                return model;
            }
            
            public void setModel(String model) {
                this.model = model;
            }
            
            public abstract void harakatlan();
        }

        public class Avtomobil extends Transport {
            @Override
            public void harakatlan() {
                System.out.println("Avtomobil yurmoqda.");
            }
        }
        
        public class Samolyot extends Transport {
            @Override
            public void harakatlan() {
                System.out.println("Samolyot uchmoqda.");
            }
        }

        public class Main {
            public static void main(String[] args) {
                Transport avtomobil = new Avtomobil();
                avtomobil.setModel("Toyota");
                System.out.println("Model: " + avtomobil.getModel());
                avtomobil.harakatlan();
                
                Transport samolyot = new Samolyot();
                samolyot.setModel("Boeing");
                System.out.println("Model: " + samolyot.getModel());
                samolyot.harakatlan();
            }
        }

Ushbu misolda, Transport abstrakt sinfi umumiy xususiyat va metodlarni o'z ichiga oladi. Avtomobil va Samolyot sinflari esa, Transport sinfidan meros olib, harakatlan metodini qayta belgilaydi (override qiladi) va o'ziga xos xatti-harakatni ifodalaydi. Abstraktsiya orqali murakkab tizimlar sodda va tushunarli shaklda taqdim etiladi.
