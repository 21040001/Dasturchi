Java - O'zgaruvchilar Turlarini O'zgartirish (Type Casting)

Java dasturlash tilida o'zgaruvchilarning turini boshqa turga o'zgartirish jarayoniga "Type Casting" deyiladi. Bu jarayon ikki turga bo'linadi:

  • Implicit (Automatic) Casting – Avtomatik o'zgarish
  • Explicit Casting – Majburiy yoki qo'lda o'zgarish

1. Implicit (Automatic) Casting – Avtomatik O'zgarish

Kichik turdagi qiymatlarni katta turga o'tkazishda avtomatik ravishda amalga oshiriladi. Ma’lumot yo‘qolishi xavfi yo‘q. Masalan, integer turidagi sonni avtomatik tarzda double turiga o'tkazish mumkin, chunki double katta hajmli ma'lumot turi hisoblanadi.

Ketma-ketlik: byte → short → int → long → float → double

int a = 100;
double b = a; // Avtomatik int → double o'tkazish

System.out.println(b); // Natija: 100.0

2. Explicit Casting – Qo'lda O'zgarish

Katta turdagi qiymatni kichik turga o'tkazishda qo‘lda amalga oshiriladi. Bu yerda ma’lumot yo‘qolishi mumkin. Masalan, double qiymatini integer turiga o'tkazganda sonning o'nlik qismi yo'qoladi.

Ketma-ketlik: double → float → long → int → short → byte

double x = 55.75;
int y = (int) x; // Qo‘lda double → int o'tkazish

System.out.println(y); // Natija: 55 (ondalik qism yo‘qoladi)

Eslatma: Katta sondan kichik songa o‘tganda qiymat chegaradan oshishi mumkin va kutilmagan natijalar chiqarishi mumkin.

int kattaSon = 300;
byte kichikSon = (byte) kattaSon; // Qiymat chegaradan oshadi

System.out.println(kichikSon); // Natija: 44 (qiymat chegaradan oshgani uchun xato)

String ↔ Son o'zgarishlari

String → son: String tipidagi raqamni haqiqiy raqamga aylantirish uchun maxsus metodlardan foydalaniladi.

String matn = "123";
int son = Integer.parseInt(matn);

System.out.println(son + 1); // Natija: 124

Son → String: Haqiqiy sonni matn ko'rinishiga o'tkazish uchun String.valueOf() metodidan foydalaniladi.

int son = 456;
String matn = String.valueOf(son);

System.out.println(matn + "78"); // Natija: "45678"

char ↔ ASCII qiymatlari

char → ASCII: Har bir belgining ASCII kodi mavjud bo‘lib, uni quyidagi usulda ko‘rish mumkin.

char belgi = 'A';
int ascii = (int) belgi;

System.out.println(ascii); // Natija: 65

ASCII → char: ASCII kodidan tegishli belgiga o'tish mumkin.

int ascii = 66;
char belgi = (char) ascii;

System.out.println(belgi); // Natija: B

O'zgarishlar uchun qisqa jadval

Turi Metod Misol
Implicit (avto) Avtomatik int → double
Explicit (qo'lda) (veriTuri) double → int
String → son Integer.parseInt() "10" → 10
Son → String String.valueOf() 10 → "10"
char ↔ ASCII (int), (char) 'A' → 65, 66 → 'B'

Eslatma va tavsiyalar

  • Explicit castingda ma'lumot yo'qolishi ehtimolini inobatga oling.
  • Son emas matnlarni sonlarga o'tkazishda NumberFormatException xatosiga duch kelishingiz mumkin.
