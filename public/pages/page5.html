O'zgaruvchilar va Ma'lumot Turlari

Java dasturlash tilida o'zgaruvchilar va ma'lumot turlari asosiy tushunchalardan biridir. Mana ular haqida batafsil tushuntirish:

O'zgaruvchilar (Variables)

O'zgaruvchilar - bu dasturda qiymatni saqlash uchun ishlatiladigan nomlangan xotira joylari. O'zgaruvchini yaratish uchun dasturchi unga ma'lum bir ma'lumot turi berishi kerak.

O'zgaruvchilarni e'lon qilish va boshlang'ich qiymat berish

        int yosh; // int turi o'zgaruvchi e'lon qilinadi
        yosh = 25; // O'zgaruvchiga boshlang'ich qiymat beriladi

        // Yoki bitta qatorda e'lon qilish va boshlang'ich qiymat berish mumkin
        int temperatura = 30;
Ma'lumot Turlari (Data Types)

Java'da ma'lumot turlari ikki turga bo'linadi: asosiy (primitive) va murojatli (reference) ma'lumot turlari.

Asosiy (Primitive) Ma'lumot Turlari

Bu turlar Java'da oldindan aniqlangan va xotirada aniq o'lchamda joy oladi. Quyida asosiy ma'lumot turlarining ba'zi misollari:

  • byte: 8-bitli o'zgaruvchi. Qiymat oralig'i -128 dan 127 gacha.
  • short: 16-bitli o'zgaruvchi. Qiymat oralig'i -32,768 dan 32,767 gacha.
  • int: 32-bitli o'zgaruvchi. Qiymat oralig'i -2^31 dan 2^31-1 gacha.
  • long: 64-bitli o'zgaruvchi. Qiymat oralig'i -2^63 dan 2^63-1 gacha.
  • float: 32-bitli o'zgaruvchi. Qaytarmas sonlar uchun (floating-point).
  • double: 64-bitli o'zgaruvchi. Qaytarmas sonlar uchun (floating-point).
  • boolean: faqat true yoki false qiymatga ega bo'lishi mumkin.
  • char: 16-bitli o'zgaruvchi. Yagona Unicode belgi.
Murojatli (Reference) Ma'lumot Turlari

Bu turlar ob'ektlarni ifodalaydi va xotirada o'zgaruvchining haqiqiy ma'lumot manzilini saqlaydi. Ular sinf, interfeys yoki massiv ko'rinishida bo'lishi mumkin. Misollar:

  • String: matnli qiymatlarni saqlash uchun ishlatiladi.
  • Massivlar: bir xil turdagi elementlar to'plami (masalan, int[], String[]).
Misol:

        public class Main {
            public static void main(String[] args) {
                // Asosiy ma'lumot turlari
                int yosh = 25;
                double narx = 19.99;
                boolean studentmi = true;
                char belgi = 'A';

                // Murojatli ma'lumot turlari
                String ism = "Ahmad";
                int[] sonlar = {1, 2, 3, 4, 5};

                // Natijalarni chiqarish
                System.out.println("Yosh: " + yosh);
                System.out.println("Narx: " + narx);
                System.out.println("Studentmi: " + studentmi);
                System.out.println("Belgi: " + belgi);
                System.out.println("Ism: " + ism);
                System.out.print("Sonlar: ");
                for (int son : sonlar) {
                    System.out.print(son + " ");
                }
            }
        }
