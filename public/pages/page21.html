üî• Java'da Switch-Case Operatoriga Batafsil Kirish

switch operatori - dasturda biror o'zgaruvchining qiymatini tekshirib, unga mos keladigan kod blokini bajarishga yordam beradi. Bu if-else bloklari ko'pligidan qutulish uchun ishlatiladi va kodni yanada o'qilishi oson va tartibli qiladi.

üß† Switch-Case Umumiy Yozilish Formati

Switch-case tuzilmasi quyidagicha ishlaydi: biror expression qiymati beriladi, bu qiymat ketma-ket case'lar bilan solishtiriladi. Agar mos kelgan qiymat topilsa, tegishli kod bajariladi.


switch (expression) {
    case value1:
        // Agar expression value1 ga teng bo'lsa, shu kod bajariladi
        break;
    case value2:
        // Agar expression value2 ga teng bo'lsa, shu kod bajariladi
        break;
    case value3:
        // Agar expression value3 ga teng bo'lsa, shu kod bajariladi
        break;
    default:
        // Agar hech bir case mos kelmasa, default bajariladi
}

üéØ Muhim Qoidalar va Cheklovlar

  • 1. Switch faqat byte, short, char, int, enum va String tipidagi ma'lumotlarda ishlaydi. (double, float mos emas)
  • 2. Har bir case qiymati o'zgarmas (constant) bo'lishi kerak.
  • 3. Agar break yozilmasa, kod bajarilishi quyidagi case'larga o'tib ketadi (fall-through deb ataladi).
  • 4. default bloki hech bir case mos kelmasa bajariladi va uni yozish majburiy emas, lekin tavsiya etiladi.
  • 5. Har bir case qiymati unikal bo'lishi shart, aks holda kompilyator xato beradi.

üêû Eng Ko'p Uchraydigan Switch-Case Xatolari

1. Break yozilmasa, Fall-Through Yuz Beradi

Agar har bir case'dan so'ng break yozilmasa, kod keyingi case'ga o'tib ketadi, bu esa kutilmagan natijalar beradi.


int x = 2;
switch (x) {
    case 1:
        System.out.println("Bir");
    case 2:
        System.out.println("Ikki");
    case 3:
        System.out.println("Uch");
}

Natija: Ikki va Uch chiqadi, chunki break yo'q.

2. Takroriy Case Qiymati (Duplicate case)

Agar ikki case bir xil qiymatga ega bo'lsa, dastur kompilyatsiya xatosi beradi.


int a = 5;
switch (a) {
    case 5:
        System.out.println("Beshlik");
    case 5: // ‚ùå Error: Duplicate case label
        System.out.println("Yana beshlik");
}

3. Noto'g'ri Tiplar (Tip Muvofiqsizligi)

Switch operatori double yoki float kabi tiplar bilan ishlay olmaydi.


double pi = 3.14;
switch (pi) { // ‚ùå Error
    case 3.14:
        System.out.println("Pi soni");
}

4. Default Blok Har Qanday Joyda Bo'lishi Mumkin

Default bloki kod ichida har qanday joyda tursa ham bo'ladi, lekin odatda oxirida yoziladi. Agar default'dan keyin break yozilmasa, quyidagi case'lar bajarilishi mumkin.


switch (x) {
    default:
        System.out.println("Default holat");
    case 1:
        System.out.println("Bir");
        break;
    case 2:
        System.out.println("Ikki");
}

5. String Tipi Bilan Switch (Java 7+)

Java 7 versiyasidan boshlab String qiymatlari bilan switch ishlatish mumkin.


String meva = "Olma";
switch (meva) {
    case "Olma":
        System.out.println("Olma tanlandi");
        break;
    case "Banan":
        System.out.println("Banan tanlandi");
        break;
    default:
        System.out.println("Noma'lum meva");
}

üìö Qo'shimcha Ma'lumot

Java switch operatori if-else ketma-ketliklariga qaraganda samaraliroq ishlaydi. Chunki kompilyator ichki darajada jump table yoki hash table yaratadi va kerakli case'ga tezda sakraydi.

üî• Super Daraja: Ko'p Case'lar Bitta Amalga Olib Boriladi

Agar bir nechta case'lar bir xil ishni bajarishi kerak bo'lsa, ularni ketma-ket yozib bitta break bilan yakunlash mumkin.


int haftaKuni = 6;
switch (haftaKuni) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        System.out.println("Ish kuni");
        break;
    case 6:
    case 7:
        System.out.println("Dam olish kuni");
        break;
    default:
        System.out.println("Xato kun");
}
